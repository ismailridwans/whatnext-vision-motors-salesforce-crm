# GitHub Pages Runbook

## Goal

Publish the interactive project website from `docs/index.html` so mentors can open a live presentation link.

## Recommended setup

1. Open the repository settings on GitHub.
2. Open the **Pages** section.
3. Set the source to **Deploy from a branch**.
4. Choose the `main` branch.
5. Choose the `/docs` folder.
6. Save the configuration.

## Expected site URL

If GitHub Pages is enabled for this repository owner, the site should be available at:

`https://ismailridwans.github.io/whatnext-vision-motors-salesforce-crm/`

## Notes

- The interactive website entry point is `docs/index.html`.
- The site uses plain HTML, CSS, and JavaScript, so no build pipeline is required.
- Keep the supporting files `docs/site.css` and `docs/site.js` in the same folder as `docs/index.html`.
